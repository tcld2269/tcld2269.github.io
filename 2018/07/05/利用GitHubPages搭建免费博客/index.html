利用GitHubPages搭建免费博客

Jul 5, 2018 | 程序人生

如果你想拥有一个个人博客，最简单的方式就是去博客平台注册一个账号，例如新浪、网易等等。但是这样的博客网址却无法自定义，而且一般会很长，无法记住。如果希望要一个自己想要的域名，一个就是自己购买服务器，另一个就是利用Github Pages。

很多人可能并不清楚github pages是干嘛的，官网是：https://pages.github.com/，其实算github提供的一个服务，我们可以免费的再其上面搭建自己的网站，也有很多人利用github pages作为自己的个人博客站点。当然，你可以搭建任意的网站，只要你喜欢！

下面要记录一下我的博客创建步骤，一则方便查阅，二则方便他人！

一.Github Pages篇

1.注册进入github.com注册一个账号。
2.新建一个仓库，起名：你的用户名.github.io
3.上传一个index.html页面
可能第三步很多人就不会了，因为你需要下载Git客户端。下载安装好后，复制你的链接，Clone到本地。

在硬盘上的任意文件夹，进入命令行，执行复制的路径，如下:

git clone https://github.com/你的用户名/你的用户名.github.io.git

默认的地址就是：
http://你的用户名.github.io

二.域名解析篇

此时虽然你的博客搭建成功，可以访问了，不过域名还是别人的，用你自己的域名的话还差一步。
1.首先你得有自己的域名（可以自己去域名供应商处购买，推荐阿里云、腾讯云、新网等等）
2.购买完成后进行域名解析，解析到你的博客的地址
3.新建CNAME文件，里面输入你需要解析的域名，例如blog.mudongzi.com
4.上传提交的你的GitHub

ok，试一下吧，你可以用自己的域名进行访问了
